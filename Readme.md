# CW to Personal

Chordwikiの曲ページからChordPro本文をワンクリックでコピーし、指定したWebサイトを新規タブで開くChrome拡張です。

> **注意**: この拡張は野良（非公開）Chrome拡張です。Chromeウェブストアには公開されていません。手動でインストールしてください。

---

## 使い方

1. 拡張をインストール（下記参照）
2. 拡張アイコンをクリックし、ポップアップの「機能を有効にする」をONにします。
3. Chordwikiの曲ページを開くと、右上に「CW → Personal」ボタンが表示されます。
4. ボタンを押すと、編集ページに自動遷移し、ChordPro本文がクリップボードにコピーされます。
5. その後、設定したWebサイトが新規タブで開きます。
6. オプションで「元のChordwikiタブを自動で閉じる」ことも可能です。

---

## ポップアップメニュー（拡張アイコン）

- 拡張アイコンをクリックするとメニューが開きます。
- 「機能を有効にする」で本機能のON/OFFを切り替えられます。
- OFF時はページ上の「CW → Personal」ボタンを表示せず、処理も実行しません。
- 「詳細設定を開く」からオプション画面を開けます。

---

## オプション設定

- 「CW → Personal」ボタン押下時に開くWebサイトのURLを設定できます。
- 「元のChordwikiタブを閉じる」ON/OFFも選択可能です。
- オプション画面は拡張の管理画面から開けます。
- 保存先は `chrome.storage.sync` です。

---

## インストール方法（手動）

1. このリポジトリをダウンロードまたはクローンします。
2. Chromeで `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」をONにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このプロジェクトのフォルダを選択します。
5. 拡張が追加されます。
